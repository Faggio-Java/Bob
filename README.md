# Bob
Bob is a Package Manager mainly for lfs systems <br/>
Usage: bob "package name" <br/>
Usage: bob list, to list all ports
<br/> <br/>
Dependencies: <br/>
[git](https://github.com/git/git) <br/>
make <br/>
[cmake](https://github.com/Kitware/CMake) <br/>
<br/>
Installation:
```
sh install.sh
```
